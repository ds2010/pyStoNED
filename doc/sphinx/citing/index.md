# Citing pyStoNED

If you use `pyStoNED` for published work, we encourage you to cite our [pyStoNED](https://arxiv.org/abs/2109.12962) paper and other related theoretical papers. Please use the following BibTeX entries:

- pyStoNED paper

```
@techreport{Dai2021,
	title    = {{pyStoNED: A Python package for convex regression and frontier estimation}},
	author   = {Dai, Sheng and Fang, Yu Hsueh and Lee, Chia Yen and Kuosmanen, Timo},
	year     = 2021,
	type     = {{arXiv preprint arXiv:2109.12962}}
}
```
